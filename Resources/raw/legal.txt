<h3>Intoduction</h3>
Simple note/todo list application with a Widget front-end.
Each note consists of a Task Name and a Note field. The Note field can be of any length.
The note can be given a Priority and a Due Date, and can be marked as Done.
The display of notes on the Widget can be sorted by any combination of Done, Priority and Due Date.
 
<h3>Version 1.0 - Known Problems and Missing Features</h3>

1) Although a Notification icon exists and can be set, the notification implementation is outstanding<br>
2) Only a single set of notes can be displayed.  It is intended for multiple sets of notes toi be availabel for display on different Widgets.<br>
